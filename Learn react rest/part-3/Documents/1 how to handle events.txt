3ï¸âƒ£ Class Component Method (Using `this` binding)
In class components, event handlers need to be explicitly bound.

```

  constructor(props) {
    super(props);
    this.handleClick = this.handleClick.bind(this);
  }

  handleClick() {
    alert("Button Clicked!");
  }

  render() {
    return <button onClick={this.handleClick}>Click Me</button>;
  }
}
```

ğŸ”¹ Avoids re-creating functions but requires explicit binding.

Arrow functions automatically bind `this`.

6ï¸âƒ£ Event Delegation using `onClick` on Parent Element
Attach an event to a parent and use `event.target` to determine the clicked element.

```

  if (event.target.tagName === "BUTTON") {
    alert(`You clicked ${event.target.innerText}`);
  }
};

<div onClick={handleClick}>
  <button>Button 1</button>
  <button>Button 2</button>
</div>
```

7ï¸âƒ£ Synthetic Event Pooling
React wraps native events inside a synthetic event system for optimization.

